# Guia de Instalação do Live Share no Cursor

## Passo a Passo para Instalar o Live Share

### Método 1: Via Interface do Cursor (Recomendado)

1. **Abra o Cursor**
2. **Acesse o Marketplace de Extensões:**
   - Pressione `Ctrl + Shift + X` (ou `Cmd + Shift + X` no Mac)
   - Ou clique no ícone de Extensões na barra lateral esquerda

3. **Procure por "Live Share":**
   - Digite "Live Share" na barra de pesquisa
   - Procure pela extensão oficial: **"Live Share"** da Microsoft (ms-vsliveshare.vsliveshare)

4. **Instale as extensões necessárias:**
   - **Live Share** (principal)
   - **Live Share Audio** (opcional, para chamadas de voz)
   - **Live Share Pack** (pacote completo)

5. **Reinicie o Cursor** após a instalação

### Método 2: Via Command Palette

1. Pressione `Ctrl + Shift + P` (ou `Cmd + Shift + P` no Mac)
2. Digite: `Extensions: Install Extensions`
3. Procure por "Live Share" e instale

## Como Usar o Live Share

### Para Iniciar uma Sessão (Hospedar):

1. Pressione `Ctrl + Shift + P` e digite: `Live Share: Start Collaboration Session`
2. Ou clique no ícone "Live Share" na barra de status (canto inferior esquerdo)
3. Uma URL será gerada - compartilhe com seu colega
4. Você pode copiar o link ou enviar por email

### Para Entrar em uma Sessão (Convidado):

1. Receba o link da sessão do host
2. Clique no link ou cole no navegador
3. O Cursor abrirá automaticamente e você entrará na sessão

## Recursos do Live Share

- ✅ Edição colaborativa em tempo real
- ✅ Compartilhamento de terminal
- ✅ Compartilhamento de servidor local (port forwarding)
- ✅ Chat integrado
- ✅ Chamadas de voz (com Live Share Audio)
- ✅ Sincronização de cursor e seleção

## Requisitos

- Ambos precisam ter o Live Share instalado
- Conta Microsoft/GitHub (para autenticação)
- Conexão com internet

## Troubleshooting

Se tiver problemas:

1. Verifique se ambos têm a extensão instalada
2. Certifique-se de estar logado com uma conta Microsoft/GitHub
3. Verifique sua conexão de internet
4. Tente reiniciar o Cursor

## Nota Importante

O arquivo `.vscode/extensions.json` foi criado com as extensões recomendadas. 
O Cursor pode sugerir a instalação dessas extensões quando você abrir o projeto.
